## 🎯 الطريقة الأسهل: Railway (7 خطوات فقط)

### ✅ الخطوة 1: إنشاء حساب GitHub
1. اذهب إلى **[github.com](https://github.com)**
2. اضغط على "Sign up"
3. اختر اسماً وكلمة مرور
4. أدخل بريدك الإلكتروني
5. فعّل الحساب عبر البريد

### ✅ الخطوة 2: رفع المشروع إلى GitHub
1. اذهب إلى: **[github.com/new](https://github.com/new)**
2. اسم المستودع: `maaref-haithem`
3. اضغط "Create repository"
4. صفحة جديدة ستظهر → اضغط على **"uploading an existing file"**
5. اسحب ملف `maaref-haithem-project.zip` إلى الصفحة
6. انتظر حتى يكتمل الرفع → اضغط "Commit changes"

### ✅ الخطوة 3: إنشاء حساب Railway
1. اذهب إلى **[railway.app](https://railway.app)**
2. اضغط "Sign in with GitHub"
3. اسمح بالدخول

### ✅ الخطوة 4: نشر التطبيق
1. اضغط على **"New Project"**
2. اختر **"Deploy from GitHub repo"**
3. اختر المستودع الذي رفعته (`maaref-haithem`)
4. ⏳ انتظر دقيقتين حتى ينتهي البناء

### ✅ الخطوة 5: إضافة قاعدة بيانات
1. في نفس الصفحة، اضغط **"New"** → **"Database"** → **"Add PostgreSQL"**
2. ⏳ انتظر 30 ثانية حتى يتم إنشاؤها
3. راح تظهر متغيرات البيئة تلقائياً

### ✅ الخطوة 6: إضافة رابط Google Sheets
1. في التبويب **"Variables"** 
2. اضغط **"New Variable"**
3. اكتب:
   - **KEY:** `NEXT_PUBLIC_GSHEET_API_URL`
   - **VALUE:** `https://script.google.com/macros/s/AKfycbxjMiWT4yVCQ3BwVcePyK-TtQM2K_Bf54ZagNMVPq4ysJW9UJgyjjjaQVfa-NUYOtqb/exec`
4. اضغط **"Add"**

### ✅ الخطوة 7: تهيئة قاعدة البيانات
1. اذهب إلى تبويب **"Deployments"**
2. اضغط على النشر الأخير
3. اضغط على **"Console"** (الطرفية)
4. اكتب الأمر التالي واضغط Enter:
```bash
npx drizzle-kit push
```
5. ⏳ انتظر حتى ينتهي (سيظهر "✓ Applied")
